
# Contributing Checklist
- mix format / Prettier
- Lints clean (Credo when enabled)
- Comments and @specs on every function
- Structured logs at new logic points
- Error codes added to docs/error-codes.md
- Tests (when added) pass
- Commits: feat:/fix:/docs:/refactor:/chore:
- PR: what changed, how to test, error codes, risks, screenshots
